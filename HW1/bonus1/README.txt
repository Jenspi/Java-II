BONUS 1:

These files are classes used for creating Shape objects (Triangle, Rectangle, and Circle) inheriting the Point2D class. Bonus 1 adds the ability to check objects equality based on their height and base values.

Compiling:
Run command "javac *.java" in the terminal/bash

Running:
No need to run any files as they don't contain main methods.

Testing:
After compiling, run the command "java org.junit.runner.JUnitCore ShapeTests" to run the JUnit tester file.
